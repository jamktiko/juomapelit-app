Pelin luominen

Kirjoita nimimerkkisi.

Kun olet klikannut Luo huone -nappia, muut pelaajat voivat liittyä koodilla huoneeseen.

Huoneessa näkyy listattuna kaikki siihen liittyneet pelaajat.

Kun kaikki ovat valmiita, voit aloittaa pelin Aloita-napilla.

Pelin valitseminen

Tällä hetkellä pelattavissa on vain Vesiputous-juomapeli. Mikäli haluat pelata tyhjällä korttipakalla muita pelejä, valitse edellisestä valikosta Pikapeli -> Korttipakka. Tyhjää korttipakkaa voi toistaiseksi pelata vain yhdeltä laitteelta.
Huone on nyt valmis! Jaa 4-kirjaiminen koodi kavereillesi ja peli voi alkaa.
